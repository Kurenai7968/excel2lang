# excel2lang

## Using

Command

``` console
flutter pub run excel2lang -e \<extension\> -f \<excel file\> -d \<saved directory\>
```

| Name             | Parameter | Default          |
| :-----:          | :----:    | :----:           |
| Target Directory | -d        | assets/languages |
| Excel File       | -f        | translation.xlsx |
| Extension        | -e        | json             |
| Dart part of     | -p        | language         |
